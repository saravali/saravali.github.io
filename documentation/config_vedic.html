Configuration: Vedic Astrology

Objects and Properties

The first page contains the configuration for astrological objects (i.e. bodies like planet, nodes) and their properties.

Most of these options can be changed on already opened charts: right click the chart and chose the option from the menu.

More detailed descriptions can be found on the Astrological Objects and Astrological Properties pages.

Vedic Chart

You can configure the display of Vedic charts on this page. The preview on the right side shows immediately the result.

Several of these options can be changed on already opened charts: right click the chart and chose the option from the menu.

Shortcuts

Keyboard

  • ↑ and ↓ (up and down arrow): change skin
  • → and ← (right and left arrow): change between east, north and south indian style
    Mouse actions
  • Right click: open context menu
  • Mouse wheel: change skin

Additional astrological information can be found on the Jaimini Astrology pages.

Display of aspects on Mouse Over

This tab allows you to configure if aspects are shown on Mouse over event (i.e. the mouse pointer is over a chart).

See Drishtis (Aspects) for information about aspects in Vedic astrology.
