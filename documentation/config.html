Configuration

The configuration dialog is the central interface for the control of the application's behaviour.

Shortcuts

The dialog can be opened with the following shortcuts

  • Keyboard: Shortcut "C"
  • Menu: Extras → Configuration
  • Toolbar: Configuration icon (wrench, normally on the right side)
  • Context Menu: Right click in the Application window list view, chose command Configuration
